# Online Grocery Store System

A comprehensive web-based online grocery store system built with Node.js, Express, and SQL Server.

## Features

- 👤 User authentication (Sign up / Login)
- 🛒 Shopping cart management
- 📦 Order placement and tracking
- 🎯 Flash deals and discounts
- 🔄 Product recommendations
- 👑 Admin panel for inventory management
- ✅ Admin verification system

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** Microsoft SQL Server
- **Authentication:** JWT, bcrypt
- **Security:** Helmet, rate limiting, SQL injection protection

## Installation

### Prerequisites
- Node.js (v14 or higher)
- SQL Server (2017 or higher)
- SQL Server Management Studio (SSMS)
